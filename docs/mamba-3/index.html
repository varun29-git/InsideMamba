1. The Compounding Error of ZOH

In Mamba-1 and Mamba-2, the continuous differential equation was discretized using Zero-Order Hold (ZOH). ZOH assumes that the input signal \(x(t)\) remains perfectly constant for the entire duration of the time step \(\Delta\).

Mathematically, this introduces an integration error margin of \(O(\Delta)\). While acceptable for short sequences, when processing contexts of 1M+ tokens, this approximation error compounds. The model's internal state slowly drifts away from the true continuous representation guaranteed by the HiPPO matrix.

2. Trapezoidal Recurrence (Crank-Nicolson)

Mamba-3 solves this drift by upgrading the integration scheme to the Trapezoidal Rule (Crank-Nicolson method). Instead of assuming the data is a flat stair-step, the model evaluates the state at both \(t-1\) and \(t\), essentially drawing a sloped line between the two points.

\[ h_t = \exp(\Delta_t A)h_{t-1} + \frac{\Delta_t}{2}\left(B_t x_t + \exp(\Delta_t A)B_{t-1}x_{t-1}\right) \]

By taking this weighted average, the approximation error drops from \(O(\Delta)\) down to \(O(\Delta^2)\). This drastically improves the model's ability to maintain high-fidelity memory over massive context windows.

3. Complex Geometric States (RoPE)

In earlier generations, the hidden state \(h_t\) consisted of real numbers (\(\mathbb{R}^N\)). The problem with real-valued exponential matrices is that they only do two things: they scale towards zero (decay) or blow up to infinity (spectral collapse).

Mamba-3 maps the latent state into the complex plane (\(\mathbb{C}^N\)). By applying a rotation matrix \(R(\theta)\) inspired by Rotary Position Embeddings (RoPE), the model can change the phase of the state vector without altering its magnitude. Like a planet in orbit, the memory state can rotate infinitely without decaying to zero or exploding, guaranteeing perfect long-term stability.

4. MIMO Rank Expansion

A standard Mamba block operates as a Single-Input Single-Output (SISO) system. If the model has 64 channels, it runs 64 entirely independent state spaces. The channels never share information. If you want to increase capacity and allow the channels to mix, the parameter count balloons quadratically \(O(D^2)\).

Mamba-3 introduces Multi-Input Multi-Output (MIMO) routing.

MIMO architecture forcing D channels through an R-dimensional bottleneck.

Instead of running independent equations, it takes all \(D\) channels and projects them into a low-dimensional bottleneck \(R\), where the data is mathematically forced to mix. It then expands this mixed data out to \(H\) heads. This allows Mamba-3 to achieve massive cross-channel interaction and representation capacity, while keeping the parameter count strictly constrained.
